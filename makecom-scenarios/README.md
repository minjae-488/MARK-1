# Make.com Scenarios

이 디렉토리는 Make.com 자동화 워크플로우의 JSON 시나리오 파일을 저장합니다.

## 📋 파일 목록

- `01-order-collection.json` - 주문 수집 시나리오
- `02-fulfillment-grouping.json` - 합배송 그룹핑 시나리오
- `03-auto-fulfillment.json` - 자동 발주 시나리오
- `04-invoice-mapping.json` - 송장 자동 등록 시나리오

## 🔧 사용 방법

1. Make.com 대시보드 접속
2. "Import Blueprint" 클릭
3. JSON 파일 업로드
4. 연결 설정 (Airtable, API 키 등)
5. 시나리오 활성화
